# 🚀 快速启动指南（必看！）

## 方法一：使用演示版（推荐，不需要数据库） ⭐

### 1️⃣ 安装依赖（如果还没安装）
```bash
cd backend
npm install

cd ../frontend
npm install
```

### 2️⃣ 启动演示版后端
```bash
cd backend
npm run demo
```

你会看到：
```
========================================
  学生校内双证管理系统 - 演示版
========================================
  服务地址: http://localhost:5000
  注意: 这是演示版本，使用内存数据
  账号: admin / admin123
========================================
```

### 3️⃣ 启动前端（新开一个终端）
```bash
cd frontend
npm start
```

### 4️⃣ 访问应用
- 浏览器打开：http://localhost:3000
- 登录账号：admin / admin123

---

## 方法二：使用真实数据库版本

### 1️⃣ 导入数据库
```bash
mysql -u root -p < db/schema.sql
# 输入密码：root
```

### 2️⃣ 启动后端
```bash
cd backend
npm start
```

### 3️⃣ 启动前端（新开一个终端）
```bash
cd frontend
npm start
```

---

## ❓ 有问题？

- 查看 [DEBUG.md](DEBUG.md)
- 查看浏览器控制台错误（F12）
- 确保后端正在 http://localhost:5000 运行
